# Qistass Pay for WooCommerce 🛒💳

بوابة الدفع الرسمية والموثوقة لمنصة **Qistass Pay** الخاصة بمتاجر ووكومرس (WooCommerce). تتيح هذه الإضافة لتجارك استقبال المدفوعات الإلكترونية بكل سهولة وأمان تام.

## ✨ المميزات (Features)
* **ربط سريع وآمن:** تكامل كامل مع API الخاص بـ Qistass Pay.
* **دعم Webhook (الإشعارات الخلفية):** تأكيد فوري للطلبات حتى لو انقطع اتصال العميل، عبر نظام تشفير عالي الأمان (HMAC-SHA256).
* **تجربة مستخدم سلسة:** تحويل آمن لصفحة الدفع وإعادة العميل للمتجر تلقائياً بعد نجاح العملية.
* **سجلات دقيقة (Logging):** نظام تتبع أخطاء مدمج لتسهيل الدعم الفني.

## 📥 طريقة التركيب (Installation)
1. قم بتحميل الإضافة كملف `ZIP` من زر **Code** > **Download ZIP** في الأعلى.
2. اذهب إلى لوحة تحكم الووردبريس الخاصة بمتجرك.
3. انتقل إلى **إضافات (Plugins)** > **أضف جديد (Add New)** > **رفع إضافة (Upload Plugin)**.
4. اختر ملف الـ `ZIP` الذي قمت بتحميله واضغط على **التنصيب الآن (Install Now)**.
5. اضغط على **تفعيل (Activate)**.

## ⚙️ الإعدادات (Configuration)
1. في لوحة تحكم ووردبريس، اذهب إلى **WooCommerce** > **الإعدادات (Settings)** > **المدفوعات (Payments)**.
2. قم بتفعيل بوابة **Qistass Pay** واضغط على زر الإعدادات الخاص بها.
3. قم بتعبئة البيانات التالية (والتي يمكنك الحصول عليها من لوحة تحكم التاجر في Qistass Pay):
   * **Merchant Number** (رقم التاجر)
   * **Public Key** (المفتاح العام)
   * **Secret Key** (المفتاح السري)
   * **Webhook Secret** (أدخل المفتاح السري هنا أيضاً لضمان التشفير)
4. اضغط على **حفظ التغييرات (Save changes)**.

## 🛡️ الأمان (Security)
تم بناء هذه الإضافة بأعلى معايير الأمان الخاصة بـ WordPress و WooCommerce. تعتمد الإضافة على نظام الـ `Zero-Trust` عبر التحقق المزدوج من العمليات باستخدام الـ API المباشر وتشفير الإشعارات.
